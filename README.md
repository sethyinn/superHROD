## ðŸ›  Core Workflow

The assistant follows a professional four-stage consultancy lifecycle:

1. **Requirement Clarification:** Conversational probing to define objectives, scope, and stakeholders.
2. **Solution Design:** Generating proposals integrated with external benchmarks and internal data.
3. **Implementation Planning:** Breaking down strategies into executable tasks and timelines.
4. **PMO Support:** Automated generation of management artifacts (WBS, RACI, Risk Registers).
